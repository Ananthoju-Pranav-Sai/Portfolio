Automated Microstress Assessment During Pregnancy Using ECG Sensing and Optimized Deep Networks

Chirag Mehta, Pranav Sai Ananthoju, Swarubini P J, and Nagarajan Ganapathy.

IEEE Sensors Letters, Vol. 8, No. 9, 2024

Proposed an end-to-end framework for detecting microecological momentary assessment (micro-EMA) based stress in pregnant mothers using preprocessed ECG signals. Extracted key physiological features and compared multiple classification pipelines, demonstrating that a 1D Convolutional Neural Network with fine-tuning (DLFT) achieves an optimal 76.50% F1-score via leave-one-subject-out cross-validation.

Agentic NPC Dialogue System

Implemented a centralized LLM service featuring agent-specific memory architectures that decouple an NPC's static identity from their dynamic, ongoing experiences. The framework utilizes vector-based retrieval mechanisms to enable highly context-aware and stateful character responses.

Validated the robustness and consistency of the dialogue service by deploying it within a text-based social deception game, requiring complex cross-agent reasoning and strict behavioral memory alignment.

Multi-Step Reasoning using Truthfulness and Faithfulness

Enhanced the Entailer architecture from open-source NLP research to generate more faithful and truthful explanations for multi-step Question Answering (QA) tasks. By systematically mitigating reasoning errors, the optimized pipeline yielded a 5–8% accuracy improvement on the OpenBookQA (OBQA) and QUARTZ datasets.

Focused on reducing belief errors within the model's generated premises. Integrated LLaMA-7B as the core engine and executed a targeted fine-tuning pipeline using the ARC and WorldTree datasets, successfully refining the model's logical deduction paths and consistency.

Algorithmic Recourse under Incomplete Causal Graphs

Tackled algorithmic recourse under incomplete causal graphs by developing a practical, data-driven method for cost-efficient, actionable interventions using only observed data to improve machine learning explainability.

Evaluated the approach on a semi-synthetic 7-variable Structural Causal Model (SCM) inspired by the UCI German Credit dataset, demonstrating its effectiveness under scenarios with partial causal knowledge.
